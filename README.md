# JEE Rank & College Predictor

A Python-based command-line application that predicts JEE Main ranks and recommends NITs/IIITs based on JoSAA cutoff data.

## Features
- Predicts CRL from JEE Main percentile
- Predicts category rank
- College Predictor (NITs & IIITs)
- Aspirational / Realistic / Safety categorization
- JoSAA Choice Filling Helpdesk

## Technologies Used
- Python
- CSV
- File Handling

## Disclaimer
Predictions are approximate and intended for reference only.
## Data Source

The opening and closing rank data used in this project is based on the official JoSAA (Joint Seat Allocation Authority) Opening & Closing Rank data.
